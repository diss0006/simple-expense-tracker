## About this project

This is my final project for **Web Development III** final assignment: **Simple Expense Tracker** built with **HTML, CSS, and JavaScript**. The goal was to deliver something simple and easy to use, a small dashboard where users can log income and expenses, see balance and totalsnupdate at a glance, and browse recent transactions in a clear list.

There is no server or database. All data lives in the browser using **`localStorage`**, so entries stay after refresh as long as the user uses the same browser and does not clear site data.

The layout follows a minimal dashboard style with summary cards at the top, a New Entry form on one side, and Transactions on the other, with color-coded amounts so income and spending are easy to tell apart. Users can delete individual rows or clear all if needed.
